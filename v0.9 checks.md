## 4. Trust DB Integration

**File:** `src/trust/mod.rs`, `src/update.rs`

### 4.1 Online behavior

- [ ] Meta check fires on every `cpac install` / `cpac trust` / `cpac audit`
- [ ] Meta check is a single lightweight GET, not a full sync
- [ ] Version hash matches → local cache used, no sync
- [ ] Version hash differs → delta sync queued for next `cpac update`
